## 🧪 テスト手順

### ステップ1: ブラウザを開く

1. ブラウザで以下のURLを開く:
   ```
   http://localhost:5173/
   ```

2. 開発者ツールを開く（F12キー）
   - Consoleタブを表示しておく

### ステップ2: ホーム画面

1. 名前を入力（例: "テストユーザー"）
2. 「ゲームを始める」ボタンをクリック
3. ロビー画面に遷移することを確認

**期待される動作**:
- ✅ ロビー画面が表示される
- ✅ モックルーム一覧が表示される

### ステップ3: Room画面へ移動

1. 画面下部のデバッグボタン「Room」をクリック
2. Room画面に遷移することを確認

**期待される動作**:
- ✅ Room画面が表示される
- ✅ ルーム情報が表示される:
  - ルームコード: ABCD1234
  - プレイヤー数: 6/6
  - 基本レート: ¥100
- ✅ プレイヤー一覧が表示される（6人）
- ✅ 「ゲーム開始」ボタンが表示される

### ステップ4: ゲーム開始

1. 「ゲーム開始」ボタンをクリック
2. Game画面に自動遷移することを確認

**期待される動作**:
- ✅ Game画面に遷移する
- ✅ ゲーム画面が表示される

### ステップ5: コンソールログ確認

ブラウザのConsoleタブで以下のログを確認:

```javascript
// ゲーム開始時
Starting game: { roomId: "room-1", players: [...], baseRate: 100 }

// ゲーム開始成功時
Game started successfully: "game-xxx"

// ゲーム状態更新時
game:started イベント受信
game:state-updated イベント受信
```

**期待されるログ**:
- ✅ `Starting game:` が表示される
- ✅ `Game started successfully:` が表示される
- ✅ エラーメッセージが表示されない

### ステップ6: バックエンドログ確認

ターミナル（バックエンドサーバー）で以下のログを確認:

```
[INFO] Game start requested { roomId: "room-1", playerCount: 6, baseRate: 100 }
[INFO] Game started successfully { roomId: "room-1", gameId: "game-xxx", playerCount: 6 }
[INFO] Broadcast to room { roomId: "room-1", eventName: "game:started" }
[INFO] Broadcast to room { roomId: "room-1", eventName: "game:state-updated" }
```

**期待されるログ**:
- ✅ `Game start requested` が表示される
- ✅ `Game started successfully` が表示される
- ✅ `Broadcast to room` が表示される（2回）
- ✅ エラーメッセージが表示されない

### ステップ7: ゲーム画面確認

Game画面で以下の要素を確認:

1. **ヘッダー**
   - ✅ ゲームタイトル「DOBON」が表示される
   - ✅ 現在のターンプレイヤーが表示される（例: "たろうのターン"）
   - ✅ 基本レート「¥100」が表示される

2. **対戦相手エリア（上部）**
   - ✅ 対戦相手5人が表示される
   - ✅ 各プレイヤーの名前が表示される
   - ✅ 各プレイヤーの手札枚数が表示される
   - ✅ 公開カードがある場合は表示される

3. **ゲームフィールド（中央）**
   - ✅ 山札が表示される（残り枚数付き）
   - ✅ 場札が表示される（カードの絵柄と数字）
   - ✅ 倍率が表示される（例: "×2"）
   - ✅ 最後にカードを出したプレイヤー名が表示される

4. **自分の手札エリア（下部）**
   - ✅ 自分の名前が表示される
   - ✅ 自分の手札が表示される（13枚）
   - ✅ カードをクリックして選択できる
   - ✅ 「X枚出す」ボタンが表示される
   - ✅ 「DOBON！」ボタンが表示される

## 🔧 トラブルシューティング

### 問題1: WebSocket接続エラー

**症状**:
```
WebSocket connection failed
Error: connect ECONNREFUSED 127.0.0.1:3000
```

**原因**: バックエンドサーバーが起動していない

**解決方法**:
```bash
cd backend
npm run dev
```

---

### 問題2: イベント送信エラー

**症状**:
```
Socket not connected
```

**原因**: WebSocket接続が確立されていない

**解決方法**:
1. ページをリロード（F5）
2. WebSocket接続が確立されるまで待つ
3. 再度ゲーム開始ボタンをクリック

---

### 問題3: ゲーム状態が更新されない

**症状**:
- ゲーム画面にモックデータが表示される
- 実際のゲーム状態が反映されない

**原因**: `game:state-updated` イベントが受信されていない

**解決方法**:
1. ブラウザのConsoleで `game:state-updated` イベントが受信されているか確認
2. バックエンドログで `broadcastGameState` が呼ばれているか確認
3. 問題が続く場合は、コードを確認

---

### 問題4: プレイヤー情報が表示されない

**症状**:
- ゲーム画面が空白
- プレイヤー情報が表示されない

**原因**: `gameState` が null

**解決方法**:
1. `game:started` イベントが正しく受信されているか確認
2. `gameStore.gameState` が更新されているか確認
3. React DevToolsで状態を確認

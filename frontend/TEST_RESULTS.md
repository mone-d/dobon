## テストコマンド

```bash
# E2Eテストを実行
npm run test:e2e

# UIモードでテストを実行（デバッグに便利）
npm run test:e2e:ui

# テストレポートを表示
npm run test:e2e:report
```

## 自動テストの利点

✅ **手動テストが不要** - コマンド1つで全テストを実行
✅ **スクリーンショット自動保存** - 失敗時の画面を確認可能
✅ **CI/CDに統合可能** - GitHub Actionsなどで自動実行
✅ **リグレッション防止** - コード変更時に既存機能が壊れていないか確認
